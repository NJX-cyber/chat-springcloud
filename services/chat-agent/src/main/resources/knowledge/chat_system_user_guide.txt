Chat 系统用户操作手册

一、这是什么

这是一个桌面端聊天系统，左侧是功能入口，右侧是具体内容区。常见入口从上到下分别是：

1. 聊天
2. 联系人
3. 设置

如果你是管理员，登录后还可以在系统托盘菜单里进入管理后台。

二、登录和注册

1. 打开应用后会先进入登录页。
2. 在“邮箱”里输入账号。
3. 在“密码”里输入密码。
4. 点击验证码图片可以刷新验证码。
5. 输入验证码后，点击“登录”。
6. 如果还没有账号，点击下方“注册账号”切换到注册模式。
7. 注册时需要填写：
   - 邮箱
   - 昵称
   - 密码
   - 确认密码
   - 验证码
8. 登录页右侧的小下拉箭头会显示本机保存过的历史邮箱，点一下可以快速填入。

三、主界面怎么理解

1. 左侧竖栏是主功能入口。
2. 聊天页显示会话列表和消息区。
3. 联系人页显示好友、群聊、新朋友等内容。
4. 设置页里可以修改个人资料、密码、文件保存路径、查看关于和更新。
5. 顶部窗口按钮可以最小化、最大化、关闭，部分页面还支持置顶。

四、更换头像

这是最常见的操作，按下面点就行：

1. 点击左侧竖栏最下方的“设置”。
2. 进入“个人信息”页面。
3. 在个人信息卡片右上角，点击“三个点”菜单。
4. 选择“修改个人信息”。
5. 在弹出的编辑页里找到“头像”一栏。
6. 点击头像框里的“选择”按钮。
7. 从本地选择一张图片，支持 jpg、png、gif、bmp 等格式。
8. 系统会自动生成头像和封面预览。
9. 检查无误后，点击“保存”。
10. 保存成功后，头像会自动刷新。

小提示：
- 如果刚换完头像没有立刻变更，可以稍等几秒或返回页面重新进入一次。
- 头像上传本质上是先选图，再由系统生成可用头像文件。

五、修改昵称、性别、地区、签名

1. 先进入“设置” -> “个人信息”。
2. 点击右上角“三个点”。
3. 选择“修改个人信息”。
4. 你可以修改这些内容：
   - 昵称
   - 性别
   - 好友权限
   - 地区
   - 个性签名
5. 填好后点击“保存”。

字段说明：
- “好友权限”打开后表示加好友不需要验证。
- “好友权限”关闭后表示加好友时需要你确认。

六、修改密码

1. 进入“设置” -> “个人信息”。
2. 点击右上角“三个点”。
3. 选择“修改密码”。
4. 按顺序输入：
   - 原密码
   - 新密码
   - 确认密码
5. 点击“修改密码”。
6. 系统会提示你重新登录。
7. 确认后，系统会自动回到登录页。

七、退出登录

1. 进入“设置” -> “个人信息”。
2. 页面下方点击“退出登录”。
3. 确认后会回到登录页。

八、聊天怎么发消息

1. 点击左侧“聊天”。
2. 在会话列表里点一个联系人或群聊。
3. 右侧会显示消息记录和输入框。
4. 在输入框里输入文字。
5. 点击“发送”按钮，或者按 `Ctrl + Enter` 发送。

聊天页常用操作：
- 在顶部搜索框里输入联系人名字或最近消息内容，可以快速筛选会话。
- 会话列表上右键，可以看到“置顶/取消置顶”和“删除聊天”。
- 往上滚动消息区顶部，会自动加载更早的历史消息。
- 左侧未读数会显示当前未读消息数量。

九、发送表情、文件、图片、视频

1. 在聊天页底部输入区上方，有表情按钮和文件按钮。
2. 点表情按钮可以插入表情。
3. 点文件按钮可以从本地选择文件发送。
4. 也可以把文件直接拖到输入区域里发送。
5. 还可以在输入框里直接粘贴截图或图片。

发送文件时注意：
- 文件过大时会提示失败。
- 不同类型文件有不同大小限制。
- 文件发送后可能先显示“正在处理”，处理完成后才算真正发送完成。

十、查看图片、视频和文件

1. 聊天消息里的图片、视频、文件都可以点击查看。
2. 图片会在独立预览窗口打开。
3. 视频会在预览窗口里播放。
4. 文件消息会显示文件名、大小和处理状态。
5. 在媒体预览窗口里，你可以：
   - 点上一张、下一张切换
   - 放大
   - 缩小
   - 还原原始大小
   - 旋转图片
   - 下载保存

如果你点的是文件消息：
- 可以直接点击下载。
- 如果是图片或视频文件，系统会优先展示封面或播放内容。

十一、如何找人

方式一：在联系人页里找

1. 点击左侧“联系人”。
2. 在左侧上方搜索框里输入名字。
3. 下面会实时筛选好友或群聊。
4. 点击结果可以直接进入聊天或详情页。

方式二：按 ID 搜索

1. 点击“联系人”。
2. 选择“找好友”。
3. 输入用户 ID 或群 ID。
4. 点击搜索图标或按回车。
5. 搜索结果出来后，按情况选择：
   - 如果是好友，点“发送消息”
   - 如果不是好友，点“加为好友”
   - 如果是群聊，点“加入群组”

十二、加好友和加群

1. 在“搜索好友”页面输入用户 ID 或群 ID。
2. 搜索到目标后，点击“加为好友”或“加入群组”。
3. 弹窗里会让你填写申请信息。
4. 填完后点“确定”。
5. 如果对方允许直接加入，可能会立即成功。
6. 如果需要对方审核，就会进入“新朋友”列表等待处理。

十三、处理好友申请

1. 点击左侧“联系人”。
2. 点击“新朋友”。
3. 进入后会看到所有待处理申请。
4. 每条申请右侧都有操作菜单。
5. 你可以选择：
   - 同意
   - 拒绝
   - 拉黑
6. 处理完成后列表会刷新。

补充说明：
- 好友申请数量会显示在“新朋友”旁边的角标上。
- 同意好友申请后，好友会自动进入“我的好友”。
- 同意群申请后，群聊会进入“我加入的群聊”。

十四、查看好友资料、发送消息、删除好友、拉黑好友

1. 点击“联系人”。
2. 在“我的好友”里点一个好友。
3. 右侧会显示好友详情。
4. 你可以在详情页中：
   - 点击“发送消息”
   - 点击“删除好友”
   - 点击“加入黑名单”

注意：
- 删除好友后，对方和你会解除好友关系。
- 拉黑后，对方会进入黑名单状态，后续关系处理会受限制。

十五、创建群聊

1. 点击左侧“联系人”。
2. 在“我的群聊”区域，点击“新建群聊”。
3. 进入群聊编辑页后，填写：
   - 群名称
   - 群头像
   - 加入权限
   - 群公告
4. 群头像点击“选择”按钮后上传图片。
5. 如果你想让别人直接加群，选择“无需管理员同意”。
6. 如果你想控制加群，选择“需要管理员同意”。
7. 填完后点击“创建群聊”。

十六、修改群聊信息

只有群主可以修改群信息。

操作步骤：

1. 进入群聊详情页。
2. 点击右上角“更多”菜单。
3. 选择“修改群信息”。
4. 可以修改：
   - 群名称
   - 群头像
   - 加入权限
   - 群公告
5. 修改后点击保存。

十七、查看群成员、添加成员、移除成员

1. 在聊天页进入某个群聊。
2. 点击聊天标题右侧的“更多”图标。
3. 会打开群聊详情抽屉。
4. 群主可以看到群成员列表。
5. 群主还能点：
   - “添加”
   - “移除”
6. 选择后会弹出成员选择框。
7. 在左侧和右侧之间移动成员。
8. 点“确定”完成操作。

小提示：
- 群主可以管理成员。
- 普通成员只能查看群信息，不能增删成员。

十八、退群和解散群

如果你是群主：

1. 进入群聊详情页。
2. 点击右上角菜单。
3. 选择“解散该群”。
4. 确认后群聊会被解散。

如果你不是群主：

1. 进入群聊详情页。
2. 点击右上角菜单。
3. 选择“退出该群”。
4. 确认后你会离开该群。

十九、群聊里快速查看信息

1. 在聊天页顶部，群名右边会显示群成员数量。
2. 点击群聊右侧的更多按钮，可以打开群聊详情。
3. 详情里能看到：
   - 群 ID
   - 群名称
   - 群成员数
   - 加入权限
   - 群公告

二十、文件保存路径怎么改

1. 点击左侧“设置”。
2. 进入“文件管理”。
3. 页面会显示当前文件保存路径。
4. 点击“修改保存路径”。
5. 选择一个新的文件夹。
6. 系统会把当前用户的文件复制到新目录。
7. 等复制完成后，保存路径会自动更新。
8. 如果你只是想打开保存目录，点击“打开文件保存位置”。

小提示：
- 第一次修改路径时，复制文件可能需要一些时间。
- 文件会按用户独立存放，不会和别人的文件混在一起。

二十一、关于版本更新

1. 点击左侧“设置”。
2. 进入“关于我们”。
3. 点击“检查更新”。
4. 如果有新版本，会弹出更新窗口。
5. 更新窗口会显示版本说明。
6. 你可以选择：
   - 稍后再说
   - 立即更新

说明：
- 如果是本地安装包，系统会下载后再安装。
- 如果是外链更新，会打开外部链接。

二十二、如果你是管理员

只有管理员账号才会看到这些功能。

1. 登录成功后，右键系统托盘里的 chat 图标。
2. 在菜单里点“管理后台”。
3. 进入后可以管理：
   - 用户管理
   - 靓号管理
   - 群组管理
   - 系统设置
   - 版本管理

管理员常见操作：

用户管理
- 搜索用户 ID、昵称、邮箱
- 启用或禁用账号
- 强制某个用户下线

靓号管理
- 新增或修改靓号账号
- 删除不需要的靓号

群组管理
- 搜索群 ID、群名、群主
- 查看群状态
- 解散群聊

系统设置
- 设置最多可创建群数量
- 设置群成员上限
- 设置图片、视频、其他文件大小限制
- 修改机器人昵称
- 修改机器人头像
- 修改机器人欢迎语

版本管理
- 新增版本
- 选择本地安装包或外链
- 编写更新内容
- 发布版本
- 删除旧版本

二十三、AI 聊天怎么用

1. 在聊天列表里找到 `Urobot`。
2. 点击进入。
3. 在下方输入问题。
4. 点击“发送”或按 `Ctrl + Enter`。
5. AI 的回答会一边返回一边显示。

二十四、常见提示

1. 发送空消息不行，系统会提示你先输入内容。
2. 文件太大时会发送失败。
3. 没有好友关系时，不能直接当普通好友聊天，需要先申请。
4. 群主才可以修改群信息、添加或移除成员、解散群聊。
5. 修改密码后必须重新登录。
6. 图片、视频、文件都可以点开查看，文件还可以下载到本地。
7. 如果某个按钮你看不到，通常是当前页面不对，或者账号权限不够。

二十五、最常用的操作路径速记

1. 更换头像：设置 -> 个人信息 -> 右上角三个点 -> 修改个人信息 -> 头像选择 -> 保存
2. 改密码：设置 -> 个人信息 -> 右上角三个点 -> 修改密码
3. 发消息：聊天 -> 选会话 -> 输入框 -> 发送 / Ctrl+Enter
4. 加好友：联系人 -> 找好友 -> 搜索 -> 加为好友 -> 填申请信息
5. 看申请：联系人 -> 新朋友 -> 同意 / 拒绝 / 拉黑
6. 建群：联系人 -> 我的群聊 -> 新建群聊
7. 管群：群聊详情 -> 更多 -> 修改群信息 / 添加成员 / 移除成员 / 退群 / 解散群
8. 改文件路径：设置 -> 文件管理 -> 修改保存路径
9. 查更新：设置 -> 关于我们 -> 检查更新

二十六、常见功能
- 登录：启动应用 → 输入邮箱/密码/验证码 → 登录
- 注册账号：登录页 → 注册账号 → 填写信息 → 注册
- 快速填入历史邮箱：登录页右侧下拉箭头 → 选择历史邮箱
- 更换头像：设置 → 个人信息 → 右上角三个点 → 修改个人信息 → 头像选择 → 保存
- 修改资料（昵称/性别/地区/签名）：设置 → 个人信息 → 右上角三个点 → 修改个人信息 → 编辑字段 → 保存
- 修改密码：设置 → 个人信息 → 右上角三个点 → 修改密码 → 输入新旧密码 → 确认（需重新登录）
- 退出登录：设置 → 个人信息 → 退出登录
- 发送文字消息：聊天 → 选择会话 → 输入框 → 发送 或 Ctrl+Enter
- 发送表情：聊天输入区上方表情按钮 → 选择表情
- 发送文件/图片/视频：文件按钮选择/拖拽/粘贴 → 发送
- 查看媒体文件：点击消息中的图片/视频 → 预览窗口操作
- 下载文件：点击文件消息 → 下载
- 搜索会话：聊天页顶部搜索框 → 输入关键字
- 会话置顶/取消置顶：会话列表右键 → 置顶/取消置顶
- 删除聊天：会话列表右键 → 删除聊天
- 查找联系人：联系人 → 搜索框输入名字
- 按ID搜索：联系人 → 找好友 → 输入ID → 搜索
- 添加好友：搜索结果 → 加为好友 → 填写申请 → 确定
- 加入群组：搜索结果 → 加入群组 → 填写申请 → 确定
- 处理好友申请：联系人 → 新朋友 → 同意/拒绝/拉黑
- 查看好友资料/管理好友：联系人 → 我的好友 → 点击好友 → 发送消息/删除好友/加入黑名单
- 创建群聊：联系人 → 我的群聊 → 新建群聊 → 填写信息 → 创建
- 修改群信息（群主）：群聊详情 → 更多 → 修改群信息 → 保存
- 管理群成员（群主）：群聊详情 → 添加/移除成员 → 选择成员 → 确定
- 退出/解散群聊：群聊详情 → 更多 → 退出该群/解散该群
- 修改文件保存路径：设置 → 文件管理 → 修改保存路径
- 检查更新：设置 → 关于我们 → 检查更新
- AI聊天：聊天列表 → Urobot → 输入问题 → 发送
- 管理后台（管理员）：系统托盘右键 chat 图标 → 管理后台
