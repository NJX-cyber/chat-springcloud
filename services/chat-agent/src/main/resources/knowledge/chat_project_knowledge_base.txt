Chat 系统使用流程说明

一、先认识界面

打开软件后，最常用的区域有 3 个：

1. 聊天
2. 联系人
3. 设置

左侧是功能入口，右侧是当前页面内容。右上角是窗口按钮，可以最小化、最大化和关闭。

二、登录和注册

1. 打开软件后先进入登录页。
2. 输入邮箱。
3. 输入密码。
4. 输入验证码。
5. 点击“登录”。
6. 如果你还没有账号，点击“注册账号”切换到注册页面。
7. 注册时依次填写邮箱、昵称、密码、确认密码、验证码。
8. 点击“注册”完成创建。
9. 登录页右侧如果有历史邮箱下拉，可以直接点以前用过的邮箱。

三、进入主界面后怎么用

1. 登录成功后会进入主界面。
2. 左边点击“聊天”可以看消息。
3. 左边点击“联系人”可以找好友、看群聊、新朋友。
4. 左边点击“设置”可以改个人资料、密码、文件保存位置。

四、修改头像

1. 点击左侧“设置”。
2. 进入“个人信息”页面。
3. 点击右上角的“更多”按钮。
4. 选择“修改个人信息”。
5. 在编辑页面找到“头像”一栏。
6. 点击“选择”按钮。
7. 选择一张本地图片。
8. 检查预览效果。
9. 点击“保存”。
10. 保存后返回个人信息页，就能看到新头像。

五、修改昵称、性别、地区、签名

1. 点击左侧“设置”。
2. 进入“个人信息”。
3. 点击右上角“更多”。
4. 选择“修改个人信息”。
5. 在页面里修改你想改的内容：
   - 昵称
   - 性别
   - 好友权限
   - 地区
   - 个性签名
6. 填好后点击“保存”。
7. 返回个人信息页确认是否修改成功。

六、好友权限怎么理解

在“修改个人信息”里有一个好友权限开关。

1. 打开时，别人加你好友时会更容易通过。
2. 关闭时，别人加你好友需要你确认。

七、修改密码

1. 点击左侧“设置”。
2. 进入“个人信息”。
3. 点击右上角“更多”。
4. 选择“修改密码”。
5. 按顺序输入：
   - 原密码
   - 新密码
   - 确认密码
6. 点击“修改密码”。
7. 确认后按提示重新登录。

八、退出登录

1. 点击左侧“设置”。
2. 进入“个人信息”。
3. 点击页面下方的“退出登录”。
4. 确认后回到登录页。

九、发消息

1. 点击左侧“聊天”。
2. 在会话列表里选择一个联系人或群聊。
3. 右侧会显示聊天内容。
4. 在下面的输入框里输入文字。
5. 点击“发送”按钮，或者按 `Ctrl + Enter` 发送。

聊天时你还可以：

1. 点击表情按钮插入表情。
2. 点击输入框右上方的搜索框，快速找到某个会话。
3. 在会话列表上右键，可以把会话置顶或删除。
4. 向上滚动消息区域，可以加载更早的聊天记录。

十、发送图片、视频和文件

1. 进入某个聊天窗口。
2. 点击输入区上方的文件按钮。
3. 从本地选择图片、视频或文件。
4. 也可以直接把文件拖到输入区。
5. 还可以把截图直接粘贴进输入框。
6. 选好后发送。

发送文件时注意：

1. 文件过大时会提示失败。
2. 文件会先显示“处理中”或“发送中”。
3. 成功后会变成“已完成”。
4. 如果失败，可以重新发送。

十一、查看图片、视频和文件

1. 在聊天消息里点击图片。
2. 图片会打开预览窗口。
3. 在预览窗口里可以放大、缩小、旋转、切换上一张和下一张。
4. 在聊天消息里点击视频。
5. 视频会在预览窗口里播放。
6. 在聊天消息里点击文件。
7. 可以查看文件名和大小。
8. 点击下载，把文件保存到本地。

十二、搜索会话

1. 点击左侧“聊天”。
2. 在会话列表上方的搜索框里输入关键词。
3. 可以按联系人名字或最近消息内容搜索。
4. 点击结果，直接进入对应聊天。

十三、找好友或找群

1. 点击左侧“联系人”。
2. 选择“找好友”。
3. 在搜索框里输入用户 ID 或群 ID。
4. 点击搜索按钮。
5. 找到结果后可以继续下一步操作。

如果搜索到的是好友：

1. 点“加为好友”。
2. 填写申请说明。
3. 点击确认。

如果搜索到的是群聊：

1. 点“加入群组”。
2. 填写申请说明。
3. 点击确认。

十四、处理新朋友

1. 点击左侧“联系人”。
2. 进入“新朋友”。
3. 查看别人发来的好友申请或加群申请。
4. 在每条申请右侧点击操作按钮。
5. 可以选择：
   - 同意
   - 拒绝
   - 拉黑
6. 处理完后列表会自动更新。

十五、查看好友资料

1. 点击左侧“联系人”。
2. 在“我的好友”里点某个好友。
3. 右侧会显示他的资料。
4. 你可以在资料页里：
   - 发送消息
   - 删除好友
   - 加入黑名单

如果你想聊天，直接点“发送消息”就会跳到对应聊天页。

十六、创建群聊

1. 点击左侧“联系人”。
2. 进入“我的群聊”。
3. 点击“新建群聊”。
4. 填写群名称。
5. 选择群头像。
6. 选择加入权限：
   - 无需管理员同意
   - 需要管理员同意
7. 填写群公告。
8. 点击“创建群聊”。

十七、查看和管理群聊

1. 在“我的群聊”里点一个群。
2. 右侧会显示群详情。
3. 你可以看到群 ID、群名、成员数、加入权限和公告。
4. 点击“发送群消息”可以直接进入群聊。

如果你是群主：

1. 点击群详情右上角“更多”。
2. 可以选择“修改群信息”。
3. 也可以选择“解散该群”。

如果你不是群主：

1. 点击群详情右上角“更多”。
2. 可以选择“退出该群”。

十八、管理群成员

只有群主可以管理成员。

1. 在群聊右上角点击“更多”。
2. 打开群成员面板。
3. 群主可以看到成员列表。
4. 点击“添加”可以把成员加进群里。
5. 点击“移除”可以把成员踢出群。
6. 在选择窗口里勾选成员后点击“确定”。

十九、修改群信息

1. 进入群详情页。
2. 点击右上角“更多”。
3. 选择“修改群信息”。
4. 进入编辑页后可以改：
   - 群名称
   - 群头像
   - 加入权限
   - 群公告
5. 修改完点击保存。

二十、文件保存路径

1. 点击左侧“设置”。
2. 进入“文件管理”。
3. 可以看到当前文件保存位置。
4. 点击“修改保存路径”。
5. 选择新的文件夹。
6. 等待系统复制文件。
7. 完成后新的保存位置会生效。
8. 如果只想打开保存目录，点击“打开文件保存位置”。

二十一、检查更新

1. 点击左侧“设置”。
2. 进入“关于我们”。
3. 点击“检查更新”。
4. 如果有新版本，会弹出更新提示。
5. 你可以选择“立即更新”或“稍后再说”。

二十二、AI 聊天

1. 在聊天列表里找到 `chat`。
2. 点击进入。
3. 在输入框里输入问题。
4. 点击“发送”。
5. AI 会一边生成一边显示回复。

二十三、管理员后台

如果你的账号有管理员权限，登录后可以进入管理员后台。

1. 右键或点托盘菜单里的“管理后台”。
2. 进入后可以做这些事：
   - 管理用户
   - 管理靓号
   - 管理群组
   - 修改系统设置
   - 发布版本

管理员常见操作：

1. 在“用户管理”里搜索用户，启用、禁用或强制下线。
2. 在“靓号管理”里新增、修改、删除靓号。
3. 在“群组管理”里查看群信息并解散群组。
4. 在“系统设置”里调整群数量、成员数量、文件大小限制、机器人昵称、欢迎语。
5. 在“版本管理”里新增版本、填写更新内容、发布版本。

二十四、常见使用顺序

1. 登录软件
2. 进入聊天或联系人
3. 找到人或群
4. 发消息、发文件、看资料
5. 修改头像或个人信息
6. 处理新朋友
7. 创建或管理群聊
8. 查看更新

二十五、快速记忆版

1. 改头像：设置 -> 个人信息 -> 更多 -> 修改个人信息 -> 头像 -> 保存
2. 改密码：设置 -> 个人信息 -> 更多 -> 修改密码
3. 发消息：聊天 -> 选会话 -> 输入 -> 发送
4. 加好友：联系人 -> 找好友 -> 搜索 -> 加好友 -> 填申请 -> 确认
5. 看申请：联系人 -> 新朋友 -> 同意/拒绝/拉黑
6. 建群：联系人 -> 我的群聊 -> 新建群聊
7. 管群：群详情 -> 更多 -> 修改群信息/退出/解散
8. 改保存路径：设置 -> 文件管理 -> 修改保存路径
9. 查更新：设置 -> 关于我们 -> 检查更新
